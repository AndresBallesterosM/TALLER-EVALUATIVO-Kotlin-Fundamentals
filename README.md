# ¿De qué trata este taller?

En este taller practiqué los temas básicos de Kotlin, como:

Variables y funciones

Condicionales (if y when)

Manejo de valores nulos

Clases y objetos

Lambdas

El trabajo tiene 4 actividades prácticas y un archivo con preguntas teóricas.

# ¿Cómo ejecutar el código?
Opción 1: En línea

Entrar a Kotlin Playground.

Copiar el código del archivo .kt.

Ejecutar con el botón Run.

Opción 2: En IntelliJ

Crear un proyecto Kotlin.

Pegar los archivos.

Ejecutar la función main().

# Reflexión personal
¿Qué fue lo más difícil?

Lo más complicado fue entender bien cómo funcionan las lambdas y cómo manejar los valores null sin que el programa falle.

¿Qué usaría en una app real?

Usaría:

Clases para organizar mejor la información.

Nullabilidad para evitar errores.

Lambdas para hacer filtros o acciones más dinámicas.

# 3 cosas que aprendí

Es mejor usar val cuando el valor no va a cambiar.

Kotlin ayuda a evitar errores con los valores nulos.

Dividir el código en funciones lo hace más ordenado.
